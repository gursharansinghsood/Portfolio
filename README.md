# вҡҷпёҸ Installation

## Clone Repository

```bash
git clone https://github.com/gursharansinghsood/Portfolio.git

cd Portfolio
```

---

## Backend

```bash
cd server

npm install

npm start
```

### server/.env

```env
PORT=5000

EMAIL_USER=your-email@gmail.com

EMAIL_PASS=your-app-password

CLIENT_URL=http://localhost:5173
```

---

## Frontend

```bash
cd client

npm install

npm run dev
```

Runs at

```
http://localhost:5173
```
